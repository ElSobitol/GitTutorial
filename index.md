# Туториал по работе с Git

## Начало работы

Для начала работы, необходимо инициализировать сам Git

Для его инициализации введите команду 

```
  git init
```

## Добавление файла

Для добавления файла в Git необходимо воспользоваться командой 

```
git add название файла
```
## Работа с удаленным репозиторием

Для клонирования внешнего репозитория на локальный ПК, введите команду ниже:

```
git clone
```

Для отправки данных в удаленный репозиторий необходимо воспользоваться командой

```
git push
```

Для стягивания данных с удаленного репозитория введите команду

```
git pull
```
